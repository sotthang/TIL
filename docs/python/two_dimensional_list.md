# 2차원 리스트

## 2차원 리스트 란

파이썬에서 행렬 처럼 가로x세로 형태로 사용이 필요한 경우 사용

리스트 안에 여러개의 리스트를 선언하여 사용

```python
x = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# [
#     [1, 2, 3],
#     [4, 5, 6],
#     [7, 8, 9]
# ]
```

### 전치

행렬에서 행과 열을 바꾸는 것

```python
a = [
    [1, 2],
    [3, 4],
    [5, 6]
]
transpose_a = [
    [1, 3, 5],
    [2, 4, 6]
]
```

코드 구현

```python
a = [
    [1, 2],
    [3, 4],
    [5, 6]
]

b = [list(x) for x in zip(*a)]
print(b)
>>> [[1, 3, 5], [2, 4, 6]]

c = list(map(list, zip(*a)))
print(c)
>>> [[1, 3, 5], [2, 4, 6]]
```
