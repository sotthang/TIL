# Text Tagging

## Text Tagging 이란

텍스트의 각 단어나 구절에 특정한 레이블을 붙이는 작업

이 레이블은 단어의 역할이나 의미를 나타냄

텍스트를 이해하고 구조화하는 기본 작업으로 기계 번역, 감정 분석, 챗봇 등에 거의 필수적적
