# Claude

## Claude 란

Anthropic에서 개발한 대규모 언어 모델(LLM)로, 안전성(Safety)과 정직성(Helpful, Honest, Harmless)을 중시하는 방향으로 설계된 대화형 AI

## 주요 특징

1. 대화형 인터페이스
   - 자연스러운 대화 형식으로 상호작용
   - 긴 문맥(Long Context) 처리에 강점
2. 다양한 활용 분야
   - 질문 답변
   - 코드 작성 및 디버깅
   - 텍스트 생성, 편집, 요약
3. 모델 패밀리
   - Fable / Opus / Sonnet / Haiku 등 용도별 모델 제공
   - API 및 Claude Code 등으로 접근 가능
